# 🔧 DevTools — 32 Free Browser-Based Developer Tools

> **https://devtoolbox-53g.pages.dev/**

32 online developer tools, all running **entirely in the browser**. No data uploads, no signup, no paywalls. Just pure client-side JavaScript.

## ✨ Features

- 🔒 **100% Client-Side** — Your data never leaves your browser
- 🆓 **Completely Free** — No premium tiers, no ads
- 📵 **Offline Support** — PWA with Service Worker
- 🌓 **Dark/Light Theme** — One-click toggle
- ⚡ **Instant Load** — Cloudflare CDN global edge network

## 🛠️ Tools

### Formatters
JSON Formatter · Code Formatter (HTML/CSS/JS) · Markdown Preview · CSV ↔ JSON

### Encoders & Decoders
Base64 · URL Encode · Unicode · HTML Entity · JWT Decoder

### Images & Colors
QR Code Generator · Image Compress · Image to Base64 · Color Picker · Palette Generator · Video to GIF · Meme Generator · ASCII Art · SVG Editor

### Text
Text Diff · Word Counter · Lorem Ipsum · Regex Tester

### Generators & Converters
UUID Generator · Password Generator · Timestamp Converter · Number Base · Unit Converter · Cron Parser · URL Parser

### Files & Productivity
PDF Merge · File Hash (MD5/SHA) · Pomodoro Timer

## 🏗️ Tech Stack

- **Vanilla JavaScript** — No frameworks, each tool is a standalone HTML file
- **Cloudflare Pages** — Hosting + global CDN
- **PWA** — Service Worker for offline access
- **CSS Custom Properties** — Theme support

## 🚀 Deploy Your Own

```bash
# Clone
git clone https://github.com/YOUR_USER/devtoolbox.git
cd devtoolbox

# Deploy to Cloudflare Pages
npx wrangler pages deploy . --project-name=devtoolbox
```
