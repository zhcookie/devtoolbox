📖 关于 DevTools

我们是什么

DevTools 是一个免费在线开发者工具箱，提供 JSON 格式化、Base64 编解码、二维码生成、PDF 合并、图片压缩等 26+ 实用工具。

所有工具均在浏览器端运行，数据不上传服务器，确保您的隐私和安全。

为什么做这个

市面上的在线工具站要么广告满天飞，要么把数据上传到后台。我们希望做一个干净、快速、安全的工具箱——所有计算在本地完成，开源透明。

技术栈

纯 HTML + CSS + JavaScript，托管于 Cloudflare Pages 全球 CDN。零框架依赖，极致加载速度。
