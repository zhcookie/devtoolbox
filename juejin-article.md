## 💡 为什么做这个

每个开发者都经历过：搜"JSON 格式化"，点进一个广告满天飞的网站，粘贴数据，心里嘀咕——**"我刚是不是把生产环境的 API 数据上传到了某个来路不明的服务器？"**

我想要的是：

1. **全部本地运行** — JSON 格式化、图片压缩、PDF 合并，全是 JavaScript 在浏览器里跑，数据从不出你的电脑
2. **无需注册** — 打开就用，用完关掉
3. **永久免费** — 没有会员，没有功能限制，托管在 Cloudflare Pages 免费套餐
4. **离线可用** — 整站是 PWA，装了 Service Worker，加载一次后没网也能用大部分工具

---

## 🏗️ 技术栈

- **原生 JavaScript** — 没 React 没 Vue，每个工具一个独立 HTML 文件，体积极小
- **Cloudflare Pages** — 托管 + 全球 CDN，Wrangler CLI 秒级部署
- **PWA** — Service Worker 支持离线使用
- **暗色/亮色主题** — CSS 变量实现，一键切换

简单的好处是：每个工具页面不到 100KB，从任何 Cloudflare 边缘节点秒开，零依赖，不会坏。
